Librería Pyal - Librería Interna Grupo Analytics
================================================

Esta librería tiene como objetivo reducir el tamaño de scripts a la hora
de realizar un proceso analítico de Minería de Datos o de Cienia de Datos.

Esta librería está en estapa de desarrollo, por lo tanto el número de funciones
es reducido. Se agradece cualquier tipo de colaboración.
